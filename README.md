Track your luggage and trips — know what you packed, where it went, and how to describe it at a lost-baggage desk.

## What it does

- **Bags** — record every bag with IATA Baggage Identification Chart fields (colour, material, handle type, wheel type, size category, locks, straps, distinguishing marks) plus photos and free-text notes.
- **Trips** — plan trips with flights, assigned bags, and a per-trip packing list. Items on the list can be marked packed as you go.
- **Packing templates** — five default templates (Weekend Getaway, Business Trip, Beach Holiday, City Break, Hiking) with suggested items and Amazon affiliate links. Clone a default template to create your own editable copy.

## Tech stack

- [FastAPI](https://fastapi.tiangolo.com/) — async web framework
- [SQLAlchemy 2 async](https://docs.sqlalchemy.org/en/20/) + [aiosqlite](https://github.com/omnilib/aiosqlite) — database
- [Jinja2](https://jinja.palletsprojects.com/) templates + [Bootstrap 5](https://getbootstrap.com/) — server-rendered UI
- [PyJWT](https://pyjwt.readthedocs.io/) — JWT session cookies
- [uv](https://github.com/astral-sh/uv) — dependency management

## Running locally

```bash
git clone https://github.com/your-username/bagtag.aero
cd bagtag.aero
uv sync
uv run uvicorn src.main:app --reload
```

Open http://localhost:8000, register an account, and start tracking.

## Development

```bash
uv sync --extra dev

```

## Configuration

Settings are read from environment variables (or a `.env` file via pydantic-settings):

| Variable | Default | Description |
|---|---|---|
| `DATABASE_URL` | `sqlite+aiosqlite:///./bagtag.db` | SQLAlchemy async database URL |
| `UPLOAD_DIR` | `static/uploads` | Directory for uploaded bag photos |
| `MAX_UPLOAD_SIZE_MB` | `10` | Maximum image upload size |
| `AUTH_JWT_SECRET` | *(required)* | Secret key for signing JWT tokens |
| `AUTH_JWT_ALG` | `HS256` | JWT signing algorithm |
| `AUTH_JWT_EXP_MINUTES` | `1440` | Token lifetime in minutes |

## Affiliate links

Default packing templates include Amazon affiliate links tagged `bagtagaero-20`. These links are preserved when you clone a template. Items you add yourself never carry affiliate links.
